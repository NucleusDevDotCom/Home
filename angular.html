Getting Started with Angular

  1. Install Angular CLI: Angular CLI (Command Line Interface) is a powerful tool that helps you create and manage Angular projects easily. You can install it globally using npm (Node Package Manager) by running the following command in your terminal:

    2. npm install -g @angular/cli

  2. Create a new Angular project: Once you have installed Angular CLI, you can create a new Angular project using the ng new command. Navigate to the directory where you want to create your project and run the following command:

    3. ng new my-app

    This will create a new Angular project with the name my-app in the current directory.

  3. Run the Angular app: After creating the Angular app, navigate to the project directory by running cd my-app and start the development server by running ng serve. This will start the app on http://localhost:4200/ and you can view your app in your browser.

  4. Write your first Angular component: Now that your app is up and running, you can start writing your first Angular component. Components are the building blocks of an Angular app. You can create a new component using the ng generate component command. For example, to create a new component called my-component, run the following command:

    5. ng generate component my-component

    This will create a new folder called my-component with all the necessary files for your component.

  5. Update your app: Once you have created your component, you can update your app to use it. Open the app.component.html file and add the following code:

    6. <app-my-component></app-my-component>

    This will add your my-component to the app.

  6. Start building your app: You can now start building your app by creating more components, services, and modules as per your requirement.
